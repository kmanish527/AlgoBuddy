## üí° How to Use

Once you have successfully completed the setup:

1.  Ensure your Spring Boot backend is running (typically on `http://localhost:8080`).
2.  Ensure your frontend is loaded in Chrome (using the `frontend/dist` folder).
3.  Go to any problem page on `leetcode.com`.
4.  Click the Algo Buddy icon in your browser's toolbar to open the assistant.
